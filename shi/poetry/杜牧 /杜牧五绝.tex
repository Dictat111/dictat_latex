\poetry{长安秋望}{
楼倚霜树外，镜天无一毫。\\
南山与秋色，气势两相高。\\
}
\poetry{独酌}{
窗外正风雪，拥炉开酒缸。\\
何如钓船雨，蓬底睡秋江。\\
}
\poetry{醉眠}{
秋醪雨中熟，寒斋落叶中。\\
幽人本多睡，更酌一樽空。\\
}
\poetry{不饮赠酒}{

与愁争底事，要尔作戈矛。\\
}
\poetry{过田家宅}{

}

佳人不忍折，怅望回纤手。\\
}
\poetry{还俗老僧}{

独寻一径叶，犹挈衲残衣。\\
}
\poetry{斫竹}{

}
\poetry{将赴湖州留题亭菊}{

}
\poetry{折菊}{

}
\poetry{哭李给事中敏}{

}
\poetry{黄州竹径鬬}{

}

\poetry{题新定八松院小石}{

}

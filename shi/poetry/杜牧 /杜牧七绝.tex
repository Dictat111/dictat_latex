\subsection{七言绝句}
\poetry{遣怀}{
落魄江南载酒行，楚腰肠断掌中轻。\\
十年一觉扬州梦，赢得青楼薄幸名。\\
}

\poetry{赠别二首 二}{
多情却似总无情，唯觉尊前笑不成。\\
蜡烛有心还惜别，替人垂泪到天明。\\
}

\poetry{赠别二首 一}{
娉娉袅袅十三余，豆蔻梢头二月初。\\
春风十里扬州路，卷上珠帘总不如。\\
}
\poetry{南陵道中}{
南陵水面漫悠悠，风紧云轻欲变秋。\\
正是客心孤回处，谁家红袖凭江楼。\\
}

\poetry{金谷园}{
繁华事散逐香尘，流水无情草自春。\\
日暮东风怨啼鸟，落花犹似堕楼人。\\
}
\poetry{子规}{
蜀地曾闻子规鸟，宣城又见杜鹃花。\\
一叫一回肠一断，三春三月忆三巴。\\
}
\poetry{过华清宫绝句三首 二}{
新丰绿树起黄埃，数骑渔阳探使回。\\
霓裳一曲千峰上，舞破中原始下来。\\
}
\poetry{过华清宫绝句三首 一}{
长安回望绣成堆，山顶千门次第开。\\
一骑红尘妃子笑，无人知是荔枝来。\\
}
\poetry{念昔游三首 三}{
李白题诗水西寺，古木回岩楼阁风。\\
半醒半醉游三日，红白花开山雨中。\\
}
\poetry{将赴吴兴登乐游原一绝}{
清时有味是无能，闲爱孤云静爱僧。\\
欲把一麾江海去，乐游原上望昭陵。\\
}
\poetry{江南春绝句}{
千里莺啼绿映江，水村山郭酒旗风。\\
南朝四百八十寺，多少楼台烟雨中。\\
}
\poetry{齐安郡中偶题二首 一}{
两竿落日溪桥上，半缕轻烟柳影中。\\
多少绿荷相倚恨，一时回首背西风。\\
}
\poetry{题乌江亭}{
胜败兵家事不期，包羞忍耻是男儿。\\
江东子弟多才俊，卷土重来未可知。\\
}
\poetry{泊秦淮}{
烟笼寒水月笼沙，夜泊秦淮近酒家。\\
商女不知亡国恨，隔江犹唱后庭花。\\
}
\poetry{题桃花夫人庙}{
细腰宫里露桃新，脉脉无言度几春。\\
至竟息亡缘底事，可怜金谷坠楼人。\\
}
\poetry{秋夕}{
红烛秋光冷画屏，轻罗小扇扑流萤。\\
天阶夜色凉如水，坐看牵牛织女星。\\
}
\poetry{瑶瑟}{
玉仙瑶瑟夜珊珊，月过楼西桂烛残。\\
风景人间不如此，动摇湘水彻明寒。\\
}
\poetry{山行}{
远上寒山石径斜，白云生处有人家。\\
停车坐爱枫林晚，霜叶红于二月花。\\
}
\poetry{赤壁}{
折戟沈沙铁未销，自将磨洗认前朝。\\
东风不与周郎便，铜雀春深鏁二乔。\\
}

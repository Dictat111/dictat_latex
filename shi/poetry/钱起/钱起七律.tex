\poetry{送马明府赴江陵}{
陶令南行心自永，江天极目澄秋景。\\
万室遥方犬不鸣，双凫下处人皆静。\\
清风高兴得湖山，门柳萧条双翟闲。\\
黄花满把应相忆，落日登楼北望还。\\
}
\poetry{卢龙塞行送韦掌记}{

陈琳书记本翩翩，料敌张兵夺酒泉。\\
圣主好文兼好武，封侯莫比汉皇年。\\
}

独惭皇鉴明如日，未厌春光向玉墀。\\
}
